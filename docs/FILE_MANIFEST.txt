HOW TO EXTEND
────────────────────────────────────────────────────────────────────────────

Adding a New Enemy:
  1. Create: enemies/MyEnemy.gd (extend EnemyBase)
  2. Create: enemies/MyEnemy.tscn
  3. Add spawn case in Level01_SmokeRealm.gd

Adding a New Level:
  1. Create: level/Level02_Name.gd (extend from Level01 pattern)
  2. Create: level/Level02_Name.tscn
  3. Link from MainMenu or Level01 completion

Adding a New Power-Up:
  1. Create: powerups/NewPower.gd (extend PowerUpBase)
  2. Create: powerups/NewPower.tscn
  3. Adjust duration and effects in script
  4. Add spawn in Level01_SmokeRealm.gd
