## 💡 Key Features

### Player Controls
- Move: A/D or Arrow Keys
- Jump: Space or W
- Double Jump: Jump again in air

### Game Elements
- **Lil Blunt:** The player (you!)
- **Enemies:** Tax Collectors, Fly Swarms, Vines, Boulders
- **Power-ups:** Blaze (speed+smoke), Big (scale+blocks), Diamond (shield)
- **Collectibles:** Coins (10pts), Ethereum Rings (50pts)
- **Boss:** The Auditor (final battle, 5-hit)

### Game Systems
- Score tracking
- Health hearts (3 max)
- Power-up timer display
- Checkpoint saves
- Scene transitions
- Audio system (music/SFX)
