# marketing — decision log (append-only)

- **2026-07-19 · Email-first, X-second, nothing else yet.** Facebook/Instagram
  (crypto ad restrictions, wrong demographic, pay-to-play reach) and TikTok
  (needs a video pipeline that doesn't exist) are deliberately excluded until
  X + email traction is proven by founder-digest numbers.
- **2026-07-19 · Compliance is architecture, not copy.** Consent required
  server-side, double opt-in gates ALL campaign mail, suppression enforced at
  the send seam (not campaign selection), CAN-SPAM footer in every template
  including referral invites to non-subscribers.
- **2026-07-19 · Personalization from play data, LLM for garnish only.**
  Digests are built from each player's real stats (rank/delta/deaths); the
  LLM writes ONE shared weekly blurb (Kimi, cached). Never per-player LLM
  calls — cost discipline is structural, not aspirational.
- **2026-07-19 · The level is a marketing surface.** Snapshot moments, secret-
  wall referral codes, Hall of Blaze: players share discoveries, not ads.
- **2026-07-19 · Anonymous funnel only.** Event names, no PII, no persistent
  cross-site IDs; wallet is pseudonymous identity where present (checklist F8).
- **2026-07-19 · Content engine is draft-only by construction.** Kimi K3
  generates; humans approve. Newsletter → AgentMail Draft w/ needs_approval;
  X posts → files a human pastes; taglines → JSON a human merges into
  GameManager.SHARE_TAGLINES. Scripts never edit game code and never send.
